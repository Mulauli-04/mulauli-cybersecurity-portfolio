Key Firewall Rules I Configured

Stateful rules + logging to secure my virtual lab

🚫

Default Deny (WAN → LAN)

All inbound traffic from the internet is blocked unless explicitly allowed. This is the golden rule of firewall security.

Action: BLOCK • Protocol: Any • Log: Enabled
🌐

Outbound Access (LAN → WAN)

Allowed HTTP, HTTPS, DNS, and NTP from internal machines. Restricted everything else for better control.

Action: ALLOW • Ports: 80, 443, 53 • NAT: Enabled
